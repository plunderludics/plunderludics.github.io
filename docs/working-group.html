plunderludics working group

We are mut moochi, Jonny Hopkins, and Gurn Group. We are game developers with a common interest in a style of game development we call 'plunderludics', meaning using existing videogames as raw material to make new videogames (think collage, remix, sampling, etc.) Our work so far has involved historical research, curation, tool development, and game design.
