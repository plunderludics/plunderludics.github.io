directory

this is a list of all the plunderludic games we're aware of. certainly incomplete... get in touch if you know any that aren't on this list!

videogames

mario_battle_no.1

(and many other mario mods by the same artist)
2000
Myfanwy Ashmore

Super Mario Clouds , Super Slow Tetris

(and many other rom hacks by the same artist)
2002, 2004
Cory Arcangel

'randomizers'

big list of challenges based on randomizing items/locations in emulated games

archipelago

a multi-game multi-player randomizer, where items are sent from one game to another

Magic Box

2020
Alistair Aitcheson

2016
AP Thomson and Bennett Foddy

Lawra Suits Clark
